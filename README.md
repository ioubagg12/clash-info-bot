# clash-info-bot

## Telegram Bot
Bot Name: @The_coc_caller_bot

### How to Run
1. Open terminal/command prompt.
2. Activate virtual environment:
   ```cmd
   .\.venv\Scripts\activate.bat
   ```
3. Start the bot:
   ```cmd
   python -m bot.main
   ```
4. In Telegram, start the bot by sending:
   ```
   /start
   ```
